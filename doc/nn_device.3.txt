nn_device(3)
============

NAME
----
nn_device - start a device


SYNOPSIS
--------
*#include <nanomsg/nn.h>*

*int nn_device (int 's1', int 's2');*


DESCRIPTION
-----------
Starts a device to forward messages between two sockets. _nn_device_ function
loops infinitely and sends and messages received from 's1' to 's2' and vice
versa.

To break the loop and make _nn_device_ function exit use
linknanomsg:nn_term[3] function.

RETURN VALUE
------------
The function loops until it hits an error. In such case it returns a negative
value and sets 'errno' to one of the values defined below.

ERRORS
------
*EBADF*::
One of the provided sockets is invalid.
*EINTR*::
The operation was interrupted by delivery of a signal.
*ETERM*::
The library is terminating.

EXAMPLE
-------

----
int s1 = nn_socket (AF_SP_RAW, NN_REQ);
nn_bind (s1, "tcp://eth0:5555");
int s2 = nn_socket (AF_SP_RAW, NN_REP);
nn_bind (s2, "tcp://eth0:5556");
nn_device (s1, s2);
----
